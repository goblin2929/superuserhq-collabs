02 — Build It

Build It

Four steps. Follow along in your terminal.

1

Create the skill folder

Skills live in .claude/skills/. Each skill gets its own folder with a SKILL.md file inside.

$ mkdir -p .claude/skills/sales-proposal
2

Write the frontmatter

Open .claude/skills/sales-proposal/SKILL.md in any text editor. Start with the header:

--- name: sales-proposal description: Use when creating a sales proposal for a potential hotel or travel partner ---

name

What you type after / to run this skill. Keep it lowercase with hyphens.

description

Tells Claude when to suggest this skill automatically. Start with "Use when..."

3

Write the skill content

Add this below the frontmatter in the same file. This is what Claude follows when you run the skill.

# Sales Proposal Generator You are a B2B sales specialist at tiket.com. Generate a professional sales proposal for the given client. The user will provide: client name and any known context about the prospect. ## Proposal Sections ### 1. Client Overview - Brief summary of the client's business and market position - Key challenges or opportunities you'll address ### 2. Proposed Solution - How tiket.com's platform solves their needs - Specific features and integrations relevant to them - Expected outcomes with concrete metrics ### 3. Pricing Structure - Present three tiers: Starter, Growth, Enterprise - List what's included at each level ### 4. Next Steps - Proposed timeline with weekly milestones - Key contacts and immediate action items ## Tone Professional but warm. Data-driven. Focus on ROI and partnership value. Use specifics — not vague promises.

Save the file. That's your entire skill.

4

Run it

Open Claude Code and try your new skill:

$ claude > /sales-proposal Marriott Indonesia

Claude reads your SKILL.md, follows your instructions, and generates a structured proposal. Try it now.

03 — Customize

Make It Yours

The skill works as-is, but you can customize it for your team's specific needs. Here are the lines to change.

Default

You are a B2B sales specialist at tiket.com.

Customized

You are a B2B sales specialist at tiket.com, focused on hotel partnerships in Southeast Asia. Our key differentiator is real-time inventory sync and 45M+ monthly active users.

Default

Present three tiers: Starter, Growth, Enterprise

Customized

Present three tiers: Partnership Pilot, Standard Partner, Strategic Alliance

Default

Professional but warm. Data-driven.

Customized

Confident and consultative. Lead with market data and case studies. Mirror the client's formality level.
